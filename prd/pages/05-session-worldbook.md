# 对话工作台 - 世界书编辑器

> **路由：** `/session` (worldbook 视图)
> **模块：** 对话
> **组件：** `WorldBookEditor` → `WorldBookHeader` + `WorldBookControls` + `WorldBookTable`

## 概述

世界书是 SillyTavern 生态中的核心概念 — 一组带有关键词触发条件的上下文条目。当对话中出现匹配关键词时，相关世界书条目会被自动注入到 LLM 的上下文中，让 AI 了解世界观设定、角色关系、地点信息等。本编辑器用于管理当前角色绑定的世界书条目。

## 字段

### 世界书条目表格

| 列名 | 格式 | 可排序 | 说明 |
|------|------|--------|------|
| 展开/折叠 | 箭头图标 | 否 | 点击展开查看条目完整内容 |
| 条目名称 | 文本 | 是 | 条目的描述性标题 |
| 关键词 | 标签组 | 否 | 触发此条目注入的关键词列表 |
| 状态 | 开关 | 否 | 启用/禁用此条目 |
| 注入位置 | 文本 | 是 | 条目在提示词中的注入位置和深度 |
| 操作 | 按钮组 | 否 | 编辑、删除 |

### 世界书条目编辑器（WorldBookEntryEditor）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 条目名称 | 文本输入 | 否 | 条目的描述性标题（不影响触发逻辑） |
| 关键词 | 标签输入 | 是 | 触发此条目的关键词列表（逗号分隔） |
| 二级关键词 | 标签输入 | 否 | 可选的额外匹配条件 |
| 内容 | 多行文本 | 是 | 注入到 LLM 上下文的实际文本 |
| 注入位置 | 下拉选择 | 否 | 在提示词中的位置（before_char / after_char / 等） |
| 注入深度 | 数字输入 | 否 | 在消息历史中向前插入的深度 |
| 启用状态 | 开关 | 否 | 是否启用此条目 |
| 匹配模式 | 下拉选择 | 否 | 关键词匹配方式（包含 / 完整匹配 / 正则） |
| 优先级 | 数字输入 | 否 | 多条目匹配时的优先级排序 |
| token 预算 | 数字输入 | 否 | 此条目的最大 token 配额 |
| 角色过滤 | 下拉选择 | 否 | 仅在特定角色对话中触发 |

## 交互

### 页面加载
- 从角色数据中加载绑定的世界书条目
- 在表格中按优先级排序显示

### 新建条目
- **触发：** 点击"新建条目"按钮
- **弹窗：** 打开空白世界书条目编辑器
- **保存：** 条目添加到表格，持久化到角色数据

### 编辑条目
- **触发：** 点击条目行的"编辑"按钮
- **弹窗：** 打开预填数据的世界书条目编辑器
- **保存：** 更新条目数据

### 删除条目
- **触发：** 点击"删除"按钮
- **确认：** 二次确认
- **成功：** 条目从表格和存储中移除

### 展开/折叠条目
- **触发：** 点击条目行的展开箭头
- **行为：** 展开显示条目的完整内容文本

### 启用/禁用条目
- **触发：** 切换条目行的状态开关
- **行为：** 切换条目的激活状态，禁用后不再参与关键词匹配

### 导入世界书
- **触发：** 点击"导入"按钮
- **弹窗：** ImportWorldBookModal（文件拖放 + 选择导入项 + 作用域选择）
- **流程：** 上传文件 → 预览条目 → 选择导入哪些 → 选择保存为角色级或全局 → 确认导入
- **支持格式：** SillyTavern 世界书 JSON 格式

### 导出世界书
- **触发：** 点击"导出"按钮
- **行为：** 将当前角色的世界书导出为 JSON 文件

### 全局世界书
- **触发：** 点击"全局世界书"按钮
- **面板：** 打开全局世界书管理面板
- **功能：** 管理跨角色共享的世界书条目

## API 依赖

| 操作 | 来源 | 触发时机 | 说明 |
|------|------|---------|------|
| 加载世界书 | `function/worldbook/info` | 视图切换 | 获取角色绑定的世界书条目 |
| 保存条目 | `function/worldbook/edit` | 编辑保存 | 持久化条目变更 |
| 删除条目 | `function/worldbook/delete` | 删除确认 | 移除条目 |

## 页面关系

- **入口：** 对话工作台视图切换按钮
- **出口：** 视图切换回聊天/预设/正则
- **数据耦合：** 世界书条目在对话生成时通过 WorldBookNode 自动匹配并注入上下文

## 业务规则

- 关键词匹配发生在 NodeFlow 管道的 WorldBookNode 阶段
- 匹配逻辑：扫描最近 N 条消息内容，如果包含条目的关键词则注入该条目
- 二级关键词为 AND 条件 — 主关键词和二级关键词都需匹配才触发
- 注入深度决定条目在消息历史中的插入位置（深度 0 = 最新消息前，深度 4 = 倒数第 4 条消息前）
- token 预算限制单个条目占用的上下文空间
- 全局世界书在所有角色对话中生效
- 世界书数据格式兼容 SillyTavern V2 世界信息规范
