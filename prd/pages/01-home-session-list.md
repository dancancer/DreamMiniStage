# 首页（会话列表）

> **路由：** `/`
> **模块：** 首页
> **组件：** `HomeContent` → `SessionList`

## 概述

系统首页，展示用户所有已创建的对话会话。用户可以从这里进入已有会话继续对话，也可以创建新会话。是用户打开应用后的第一个界面。

## 字段

### 会话卡片

| 字段 | 类型 | 说明 |
|------|------|------|
| 角色头像 | 图片 | 会话关联角色的头像，无头像时显示首字母 |
| 角色名称 | 文本 | 角色卡名称 |
| 最后消息摘要 | 文本 | 最近一条消息的预览文本 |
| 最后活跃时间 | 日期 | 显示相对时间（如"2小时前"） |

### 会话编辑弹窗

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 会话名称 | 文本输入 | 否 | 自定义会话显示名称 |

## 交互

### 页面加载
- 自动从 IndexedDB 加载所有会话列表
- 按最后活跃时间倒序排列
- 无会话时显示空状态引导（"开始你的第一个冒险"）

### 点击会话卡片
- **触发：** 点击任意会话卡片
- **行为：** 导航到 `/session?id={sessionId}`，进入对话工作台

### 创建新会话
- **触发：** 通过左侧导航栏的"新冒险"按钮
- **行为：** 跳转到角色卡库（`/character-cards`），选择角色后自动创建会话

### 编辑会话
- **触发：** 长按或右键会话卡片，选择"编辑"
- **弹窗内容：** 会话名称输入框
- **验证：** 无强制规则
- **成功：** Toast 提示"保存成功"，列表刷新
- **取消：** 关闭弹窗，无变更

### 删除会话
- **触发：** 长按或右键会话卡片，选择"删除"
- **确认：** 弹出二次确认对话框："您确定要删除这个会话吗？"
- **成功：** Toast 提示，会话从列表移除
- **失败：** Toast 提示"删除会话失败"

## API 依赖

| 操作 | 类型 | 来源 | 触发时机 | 说明 |
|------|------|------|---------|------|
| 获取会话列表 | IndexedDB | `useSessionStore` | 页面加载 | 读取本地所有会话记录 |
| 删除会话 | IndexedDB | `useSessionStore` | 点击确认删除 | 删除指定会话及其对话数据 |
| 更新会话 | IndexedDB | `useSessionStore` | 编辑保存 | 更新会话名称等元数据 |

## 页面关系

- **入口：** 应用启动默认页面；404 页面"返回首页"链接
- **出口：** 
  - 点击会话 → `/session?id={id}`（对话工作台）
  - 左导航"角色卡" → `/character-cards`（角色卡库）
  - 左导航"人格" → `/personas`（人格管理）
- **数据耦合：** 会话列表与 `useSessionStore` 共享状态；在对话工作台中创建新会话后，返回首页会自动刷新列表

## 业务规则

- 会话数量无上限，但受浏览器 IndexedDB 存储限制
- 删除会话同时清除该会话的全部对话历史
- 会话排序固定为最后活跃时间倒序，不支持自定义排序
