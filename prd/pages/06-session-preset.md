# 对话工作台 - 预设编辑器

> **路由：** `/session` (preset 视图)
> **模块：** 对话
> **组件：** `PresetEditor` → `PresetHeader` + `PresetControls` + `PresetTable`

## 概述

预设是控制 AI 行为的提示词模板系统。每个预设包含一组有序的提示词条目（如系统提示、角色设定、对话风格指令等），这些条目按顺序组装成发送给 LLM 的 messages 数组。用户可以创建、编辑、切换预设来调整 AI 的行为风格。

## 字段

### 预设选择器

| 字段 | 类型 | 说明 |
|------|------|------|
| 预设下拉 | 下拉选择 | 切换当前活跃预设 |

### 提示词条目表格

| 列名 | 格式 | 说明 |
|------|------|------|
| 选择框 | 复选框 | 批量操作选择 |
| 序号 | 数字 | 条目在预设中的排列顺序 |
| 名称 | 文本 | 提示词条目的标识名称 |
| 角色 | 标签 | 消息角色（system / user / assistant） |
| 启用 | 开关 | 是否包含此条目 |
| 操作 | 按钮 | 编辑、上移/下移排序 |
| 展开内容 | 文本 | 点击展开查看完整提示词文本 |

### 提示词编辑弹窗（EditPromptModal）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | 文本输入 | 是 | 条目标识名称 |
| 角色 | 下拉选择 | 是 | system / user / assistant |
| 内容 | 多行文本 | 是 | 提示词文本，支持宏变量（如 `{{char}}`、`{{user}}`） |
| 启用 | 开关 | 否 | 默认启用 |

### 新建/复制预设弹窗

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 预设名称 | 文本输入 | 是 | 新预设的名称 |

## 交互

### 页面加载
- 加载当前活跃预设的所有提示词条目
- 条目按序号排列显示在表格中

### 切换预设
- **触发：** 在预设下拉列表中选择不同预设
- **行为：** 刷新表格，显示新预设的条目
- **影响：** 下次对话生成时使用新选中的预设

### 新建预设
- **触发：** 点击"新建预设"按钮
- **弹窗：** 输入预设名称
- **结果：** 创建空预设，自动切换到新预设

### 复制预设
- **触发：** 点击"复制预设"按钮
- **弹窗：** 输入新预设名称
- **结果：** 基于当前预设创建副本

### 重命名预设
- **触发：** 点击"重命名"按钮
- **弹窗：** 修改预设名称

### 删除预设
- **触发：** 点击"删除"按钮
- **确认：** 二次确认
- **限制：** 无法删除最后一个预设

### 导入预设
- **触发：** 点击"导入"按钮
- **弹窗：** ImportPresetModal（文件拖放）
- **支持格式：** SillyTavern 预设 JSON 格式

### 导出预设
- **触发：** 点击"导出"按钮
- **行为：** 将当前预设导出为 JSON 文件

### 编辑提示词条目
- **触发：** 点击条目的"编辑"按钮
- **弹窗：** EditPromptModal，编辑名称、角色、内容
- **保存：** 更新条目，表格刷新

### 启用/禁用条目
- **触发：** 切换条目的启用开关
- **行为：** 禁用的条目不会被包含在发送给 LLM 的 messages 中

### 调整条目顺序
- **触发：** 点击上移/下移按钮
- **行为：** 调整条目在预设中的排列顺序
- **影响：** 顺序决定了 messages 数组的组装顺序

### 展开/折叠条目
- **触发：** 点击条目行
- **行为：** 展开/折叠显示完整提示词文本

## API 依赖

| 操作 | 来源 | 触发时机 | 说明 |
|------|------|---------|------|
| 加载预设 | `function/preset/*` | 视图切换 | 获取预设列表和条目 |
| 编辑条目 | `function/preset/edit` | 编辑保存 | 更新/删除/切换条目 |
| 导入预设 | `function/preset/import` | 导入确认 | 解析并存储预设文件 |
| 导出预设 | `function/preset/download` | 点击导出 | 序列化预设为 JSON |
| 全局预设 | `function/preset/global` | 全局操作 | 管理全局预设 |

## 页面关系

- **入口：** 对话工作台视图切换按钮
- **出口：** 视图切换回聊天/世界书/正则
- **数据耦合：** 预设在对话生成时通过 PresetNode 组装 messages 数组

## 业务规则

- 预设区分角色级（绑定当前角色）和全局级（跨角色）
- 提示词支持宏变量替换：`{{char}}`（角色名）、`{{user}}`（用户名）、`{{persona}}`（人格描述）等
- 条目顺序直接影响发送给 LLM 的 messages 数组组装顺序
- 特殊条目类型：Chat History 标记 — 标识对话历史消息的插入位置
- 预设数据格式兼容 SillyTavern 预设规范
