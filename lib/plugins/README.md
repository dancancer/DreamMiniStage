# plugins/

插件系统。管理第三方插件的发现、注册与执行。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `plugin-registry.ts` | 核心 | 插件注册表 |
| `plugin-discovery.ts` | 核心 | 插件发现机制 |
| `plugin-api-factory.ts` | 核心 | 插件 API 工厂 |
| `plugin-event-emitter.ts` | 核心 | 插件事件发射器 |
| `plugin-config-storage.ts` | 核心 | 插件配置存储 |
| `plugin-types.ts` | 类型定义 | 插件类型定义 |
