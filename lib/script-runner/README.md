# script-runner/

脚本运行器。在沙箱环境中执行用户脚本。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `executor.ts` | 核心 | 脚本执行引擎 |
| `sandbox-context.ts` | 核心 | 沙箱上下文环境 |
| `event-emitter.ts` | 核心 | 脚本事件发射器 |
| `global-event-bridge.ts` | 核心 | 全局事件桥接 |
| `message-bridge.ts` | 核心 | 消息桥接通信 |
| `tavern-helper.ts` | 工具 | ST 兼容辅助函数 |
| `script-storage.ts` | 核心 | 脚本存储管理 |
