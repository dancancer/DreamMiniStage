# handlers/

命令处理器集合。各类斜杠命令的具体实现。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `core.ts` | 处理器 | 核心命令（echo、if、pipe 等） |
| `events.ts` | 处理器 | 事件触发命令 |
| `generation.ts` | 处理器 | 生成相关命令 |
| `js-slash-runner.ts` | 处理器 | JS 脚本执行命令 |
| `messages.ts` | 处理器 | 消息操作命令 |
| `operators.ts` | 处理器 | 运算符命令 |
| `variables.ts` | 处理器 | 变量操作命令 |
