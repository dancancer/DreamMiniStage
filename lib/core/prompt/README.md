# prompt/

提示词处理模块。负责提示词组装、排序、后处理及格式转换。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `index.ts` | 模块入口 | 导出提示词处理功能 |
| `manager.ts` | 核心 | 提示词管理器 |
| `post-processor.ts` | 核心 | 提示词后处理器 |
| `sorting.ts` | 核心 | 提示词排序逻辑 |
| `preset-utils.ts` | 辅助工具 | 预设工具函数 |
| `converters/` | 子目录 | 格式转换器集合 |
