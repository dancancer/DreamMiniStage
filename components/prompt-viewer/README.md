# prompt-viewer/

提示词查看器组件。调试和查看发送给 LLM 的完整提示词。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `index.ts` | 模块入口 | 导出查看器组件 |
| `mobile.ts` | 入口 | 移动端入口 |
| `styles.ts` | 样式 | 组件样式定义 |
| `PromptViewerModal.tsx` | 主组件 | 查看器弹窗 |
| `PromptViewerButton.tsx` | 子组件 | 打开按钮 |
| `PromptContent.tsx` | 子组件 | 内容展示 |
| `SearchToolbar.tsx` | 子组件 | 搜索工具栏 |
| `SearchToolbarExample.tsx` | 子组件 | 搜索示例 |
| `ImageGallery.tsx` | 子组件 | 图片画廊 |
| `PromptViewerErrorBoundary.tsx` | 子组件 | 错误边界 |
